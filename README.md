# a s t r a l  -  s e a r c h
Astral is a FOSS (Free and Open Source) Private, Secure and Free-to-use Search engine. As you may know, Telemetry and Tracking is a growing issue, and our project aims to take it down. Built by the people, for the people. *Per aspera ad astra* -Seneca

This is currently a small project led by 3 people (as of writing). We wish to achieve a search engine that is fully open source, customisable to any extent and privacy first, this would mean you have to opt in for AI overviews.
